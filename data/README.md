# 示例数据目录结构

这个目录演示了PSD项目管理系统的推荐目录结构。

## 目录说明

每个项目文件夹代表一个设计项目，项目下包含多个任务文件夹。每个任务文件夹可以包含：

- `README.md` - 任务说明和元数据
- `.psd` 文件 - Photoshop设计文件
- 其他相关文件

## README.md 格式

任务的README.md文件支持frontmatter格式，例如：

```markdown
---
status: active
priority: high
prompt: "设计一个现代风格的登录页面，包含用户名、密码输入框和登录按钮"
deadline: "2023-12-31"
assignee: "设计师姓名"
---

# 任务详情

这里是任务的详细说明...

## 设计要求

- 响应式设计
- 支持深色模式
- 遵循品牌设计规范

## 更新日志

- 2023-12-20: 创建初始设计
- 2023-12-21: 添加响应式布局
```

## 状态说明

- `active`: 进行中
- `pending`: 待开始  
- `completed`: 已完成
- `paused`: 已暂停
