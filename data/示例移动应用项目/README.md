# 移动应用项目

这是一个移动应用的设计项目，包含多个功能模块的界面设计。

## 项目目标

创建一个现代、用户友好的移动应用界面，支持以下功能：
- 用户注册和登录
- 主界面导航
- 个人资料管理
- 设置页面

## 设计规范

- 遵循Material Design指导原则
- 支持深色和浅色主题
- 响应式设计，适配不同屏幕尺寸
