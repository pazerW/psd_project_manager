# 登录页面设计

## 需求说明

设计移动应用的登录页面，需要包含以下元素：

### 必需元素
- [ ] 应用Logo
- [ ] 用户名输入框
- [ ] 密码输入框
- [ ] 登录按钮
- [ ] 忘记密码链接
- [ ] 注册入口

### 设计要求
- 简洁现代的视觉风格
- 符合品牌色彩规范
- 良好的视觉层次
- 考虑错误状态显示

## AI提示词
设计一个移动端登录页面，采用简洁现代的风格，主色调使用蓝色，背景为渐变色，输入框采用圆角设计，登录按钮要醒目，整体布局要有良好的视觉平衡。

## 设计尺寸

- 安全区域考虑：顶部44px，底部34px
