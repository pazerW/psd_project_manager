# 打卡宣传图

海报尺⼨：750x1334

**文案内容**

**小标签：**学习表彰

**主标题：**阅读小超人

**学习数据部分**：

已坚持学习 xxxx天 （配置数据空着就行）

累计阅读 xxxx字（配置数据空着就行）

**辅助文案：**精读千卷书，心藏大世界

**扫码区域文案：（左边扫码文案 右边空着二维码位置）**

邀你和我一起读经典，看世界

5日阅读提升计划 0元领

提升学习力·思考力·表达力

设计要求

1.聚焦学习/阅读激励场景，海报主要体现学习相关的成就展示类物料（如学习时长、阅读量的累计记录、学习称号授予等）。

2.趣味粗描边风格，线条粗犷、手绘感强，风格活泼可爱。

3.需要结合ip（叫叫）进行设计。

### AI 思路

这里为您提供三个不同侧重点的设计思路和对应的AI绘画提示词（Prompt），旨在满足您的所有需求：750x1334尺寸、趣味粗描边风格、包含叫叫IP、以及展示学习成就的所有文案元素。

### 设计思路综述

核心策略是将“叫叫”IP形象化身为“阅读小超人”，并将枯燥的数据（天数、字数）转化为视觉上的成就勋章或能量源。

- **风格关键词（Style Keywords）：** Bold thick outlines, hand-drawn cartoon feel, playful, cute, vibrant colors, sticker art aesthetic. (趣味粗描边、手绘感卡通、活泼、可爱、鲜艳色彩、贴纸艺术美学)
    
- **IP形象（IP Character）：** 叫叫（Jojo），通常是一只戴眼镜或有学者风范的小黄鸡。在设计中需要给它穿上“小超人”的装备（如披风、眼罩）。
    
- **场景构建（Scene Building）：** 围绕“书本”、“成长”、“世界”、“飞行”展开。

**视觉元素：**

- **主角：** 戴着红色超人披风和飞行眼镜的叫叫，自信地向上飞行。

### 使用建议：

1. **IP一致性：** 在生成图像后，请务必确认画面中的黄色小鸡形象是否符合“叫叫”IP的标准设定。如果AI生成的不够准确，可能需要垫图（Image Prompt）或者后期局部重绘（Inpainting）来修正角色的脸部特征。
    
2. **文字层级：** AI生成的图像中文字可能不准确或需要替换。建议将AI生成的图像作为底图，然后在Photoshop等软件中重新排版清晰的中文标题、数据和底部文案，以确保信息传达的准确性。
    
3. **数据占位符：** 提示词中使用了“XXXX”作为数据的占位符，这在生成的图像中可能会显示为乱码或空白区域。这正是我们需要的，方便后期填入真实数据。

收到新的需求。加入“成长”、“陪伴”和“一起生活”的元素后，设计需要从单纯的“高光时刻展示”转向更有温度的“共同经历记录”。

这要求我们在画面中不仅仅展示“叫叫”IP，最好能引入一个代表用户的**儿童形象**与叫叫互动，体现他们是在日常生活中一起努力、一起成长的伙伴。

### 思路方案一：书香养成记 (侧重：陪伴成长与日常耕耘)

设计理念：

将阅读比喻成种植。叫叫不再高高在上，而是像一个小花匠，和代表用户的孩子一起给“知识之树”浇水、施肥。数据（天数和字数）是这棵树结出的硕果。这个场景非常生活化，强调了日复一日的陪伴和共同努力带来的成长。

**视觉元素：**

- **核心互动：** 戴着小草帽的叫叫和穿着背带裤的小朋友，手里拿着水壶和铲子，正在照顾一棵巨大的、由书本堆叠生长而成的魔法树。
    
- **成长表现：** 树上挂着两个醒目的果实状牌子，分别写着“已坚持学习xxxx天”和“累计阅读xxxx字”。树干上可以画刻度线，暗示长高了。
    
- **生活气息：** 地上散落着小书包、玩具、 watering can，背景是一个温馨的小院子或阳台。

### 思路方案二：并肩探索的旅程 (侧重：共同经历与里程碑)

设计理念：

将阅读具象化为一场长途旅行。重点在于“一起走过的路”。叫叫和孩子并肩前行，背着行囊，回望过去。数据是旅途中的里程碑路牌，记录着他们共同到达的地方。

**视觉元素：**

- **核心互动：** 叫叫和孩子手牵手或并肩走在一条蜿蜒的、由书页铺成的小路上。他们背着旅行背包，脸上带着满足的笑容回头看。
    
- **成长表现：** 路上竖立着两个巨大的、可爱的路牌或里程碑石头。一个写着“坚持学习xxxx天”，另一个写着“累计阅读xxxx字”。路延伸向远方，暗示未来还有更多旅程。
    
- **生活气息：** 路边有他们露营过的小帐篷、野餐垫，天空中飞过他们一起看过的热气球。

### 思路方案三：温馨的阅读角 (侧重：居家生活与潜移默化的成长)

设计理念：

最直接地体现“生活在一起”。场景设定在温馨的家中，强调阅读已融入日常生活。成长数据被巧妙地融入家庭装饰中（如墙上的身高墙贴）。

**视觉元素：**

- **核心互动：** 在一个舒适的沙发或地毯搭建的“秘密基地”里，叫叫和孩子依偎在一起看书。叫叫可能戴着阅读眼镜，正在给孩子指书上的内容。场景非常温馨、居家。
    
- **成长表现：** 背景墙上贴着一个可爱的卡通身高量尺（成长墙贴）。量尺上的刻度不是身高，而是数据节点。两个大大的卡通贴纸标记着当前的成就：“xxxx天达成！”和“xxxx字达成！”。
    
- **生活气息：** 周围有散落的抱枕、马克杯、闹钟、画笔，充满了家的烟火气。

### 执行建议

- **角色的加入：** 引入儿童角色是体现“陪伴”的关键。确保AI生成角色的风格与叫叫IP和整体粗描边风格保持一致。
    
- **情感传递：** 在选择方案后，请注意角色的表情。他们应该是开心的、互动的、有眼神交流的，这样才能传递出温暖的陪伴感。
    
- **后期合成：** 同样地，AI主要负责生成带有氛围感的底图和角色插画。所有的文案（包括标题、数据占位符、底部文案）建议在Photoshop中进行精确排版，以保证清晰度和层级感。
