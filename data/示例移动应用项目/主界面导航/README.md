# 主界面导航设计

## 需求说明

设计移动应用的主界面，作为用户登录后的首页。

### 功能要求
- [ ] 底部导航栏（4-5个主要功能）
- [ ] 顶部状态栏
- [ ] 主要内容区域
- [ ] 用户头像和快捷操作

### 导航项目
1. 首页 - 主要内容展示
2. 发现 - 探索新功能
3. 消息 - 通知和聊天
4. 我的 - 个人中心

## 设计要求
- 保持与登录页面的视觉一致性
- 清晰的信息层级
- 易于点击的交互区域
- 考虑不同状态的显示

## 尺寸规范
- 底部导航栏高度：80px
- 图标尺寸：24x24px
- 文字大小：12px
